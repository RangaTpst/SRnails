\doxysubsection{Detailed Description}
Classe de base pour les contrôleurs.

Cette classe est responsable de rendre les vues de manière dynamique en extrayant les données passées dans les vues et en incluant le fichier de vue correspondant.

\textbackslash{}\+Controllers

Contrôleur principal pour la page d\textquotesingle{}accueil.

Cette classe est responsable de l\textquotesingle{}affichage de la page d\textquotesingle{}accueil de l\textquotesingle{}application. Elle hérite de la classe {\ttfamily Base\+Controller} et utilise sa méthode {\ttfamily render} pour afficher la vue d\textquotesingle{}accueil avec les données appropriées.
